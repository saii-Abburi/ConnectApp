## 🚀 Features

- **Team Formation & Management**
  - Create and join teams
  - Manage team members and roles
  - Track team projects and contributions

- **Hackathon Platform**
  - Browse upcoming hackathons
  - Register for events
  - Track participation and achievements

- **Community Posts**
  - Share updates and achievements
  - Engage with other students
  - Stay updated with tech news

- **Project Contributions**
  - Track individual contributions
  - Showcase project work
  - Collaborate with team members

## 🛠️ Tech Stack

- **Frontend**
  - Next.js 13
  - TypeScript
  - Tailwind CSS
  - Framer Motion
  - Lucide Icons

- **Styling**
  - Modern, premium UI
  - Responsive design
  - Smooth animations
  - Dark/Light themes

## 🏃‍♂️ Getting Started

1. **Clone the repository**
bash
git clone https://github.com/yourusername/connect-platform.git
cd connect-platform

2. **Install dependencies**

```bash
npm install
# or
yarn install
```

3. **Run the development server**
```bash
npm run dev
# or
yarn dev
```

4. **Open [http://localhost:3000](http://localhost:3000) in your browser**
